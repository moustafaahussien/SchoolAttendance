نظام تحضير الطلاب

تعرّف تلقائي على وجوه الطلاب وتسجيل الحضور مرتبط بـ Google Drive

إعداد Google OAuth (مرة واحدة فقط):
  1. افتح console.cloud.google.com
  2. أنشئ مشروعاً جديداً ← فعّل Google Drive API
  3. APIs & Services → Credentials → Create → OAuth 2.0 Client ID
  4. النوع: Web application
  5. Authorized JavaScript origins: http://localhost:8000
  6. انسخ الـ Client ID وألصقه أدناه
تشغيل الملف:
افتح Terminal في نفس مجلد الملف وشغّل:
python3 -m http.server 8000
ثم افتح: http://localhost:8000/attendance-system.html
